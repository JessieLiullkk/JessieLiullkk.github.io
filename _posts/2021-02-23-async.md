# 同步和异步
## js是单线程的，同时只能有一个任务在主线程允许，而像api请求这种，需要等到请求完成拿到数据后才能继续向下执行的任务成为`异步任务`，在请求未完成前，将任务放到`任务队列`中，等到请求完成，再通过回调等方式通知主线程。

# promise
- pending
- fulfilled
- rejected

## promise 对象是一个构造函数
```markdown
const promise = new Promise(function(resolve, reject) {
  // ... some code

  if (/* 异步操作成功 */) {
    resolve(value);
  } else {
    reject(error);
  }
});
```
